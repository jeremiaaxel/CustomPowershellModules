# GitAliasesModule
Shortcut to some commonly used (by me of course) git commands.

## Git Commands
| Alias   | Command         |
|---------|-----------------|
| `gcl`   | `git clone`     |
| `gst`   | `git status`    |
| `ga`    | `git add`       |
| `gaa`   | `git add .`     |
| `gcmt`  | `git commit`    |
| `gcmsg` | `git commit -m` |
| `gpsh`  | `git push`      |
| `gpl`   | `git pull`      |
| `gco`   | `git checkout`  |
| `gb`    | `git branch`    |
| `grst`  | `git restore`   |
| `glog`  | `git log`       |
| `gdf`   | `git diff`      |

## Adding More
Edit the GitAliasesModule.psm1; create  a new function block and the exports.\
\* Some alias may conflict with the existing PowerShell commands.
